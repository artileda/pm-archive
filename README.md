## Kartini

This a package manager inspired by crux, kiss linux package manager.
so i build Kartini.

### Setup

First, set environment variable.
```sh
export KARTINI_ROOT="/test" # put your desired root system path
export KARTINI_PATH=""
export KARTINI_CACHE=""
```
Then,

``` 
./kartini help
```
